# Nipoppy

A framework for standardized organization and processing of neuroimaging-clinical datasets.

## Developer setup

Clone this repo:

```bash
git clone https://github.com/neurodatascience/nipoppy.git
```

Install `nipoppy` with dev dependencies:
```bash
cd nipoppy
pip install -e .[dev]
```

Set up `pre-commit`:
```bash
pre-commit install
```
